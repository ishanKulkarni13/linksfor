# LinksFor

LinksFor is a web application for managing and sharing link trees, built with Next.js and Tailwind CSS.

## Features
- User authentication
- Link tree creation and editing
- Admin dashboard
- Image uploading
- Customizable themes

## Tech Stack
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Node.js](https://nodejs.org/)

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
	```sh
	git clone https://github.com/kakashiB2k/linksfor.git
	cd linksfor
	```
2. Install dependencies:
	```sh
	cd client
	npm install
	# or
	yarn install
	```
3. Create a `.env.local` file in the `client` directory and add your environment variables as needed.

### Running the Development Server

```sh
cd client
npm run dev
# or
yarn dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## Project Structure
- `client/` - Main Next.js frontend
- `src/` - Source code (components, hooks, lib, etc.)
- `public/` - Static assets
